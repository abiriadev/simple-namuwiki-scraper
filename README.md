# Namuwiki scraper

Minimal scraper for [Namu Wiki](https://namu.wiki).

## Folder structure

-   `.cache`: internal cache directory
-   `w`: namuwiki article files
-   `logs`: log directory

## Usage

```sh
$ git clone https://github.com/abiriadev/simple-namuwiki-scraper && cd simple-namuwiki-scraper
$ go run .
```
